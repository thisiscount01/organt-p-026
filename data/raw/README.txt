■ 실데이터 수동 다운로드 방법 (배포 전 필수)
  1. https://www.data.go.kr 회원가입 및 로그인
  2. 아래 데이터셋 검색 후 CSV 다운로드:
     · 전국일반음식점표준데이터 (데이터셋 ID: 15096283)
       - 포함 정보: 사업장명, 주소, 영업상태, 인허가일자, 폐업일자
     · 행정안전부_일반음식점 (데이터셋 ID: 15045016)
       - 포함 정보: 위반 이력, 처분 내용, 행정처분 종류
  3. 다운로드한 CSV 파일을 이 폴더(data/raw/)에 저장
  4. node preprocess.js 실행 → data/hygiene.json / features.json 생성

■ 데이터 스키마 (CSV 헤더 기준)
  사업자등록번호, 사업장명, 소재지전체주소, 영업상태명,
  인허가일자, 폐업일자, 처분일자, 위반내용, 처분종류명
